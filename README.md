# Student Grievance Management System

A full-stack web application that enables students to submit, track, and manage grievances, while allowing administrators and investigators to review, analyze, and resolve issues efficiently.

---

## Features

### Student Panel
- Secure login and authentication  
- Submit grievances with detailed descriptions  
- Track complaint status (Pending / In Progress / Resolved)  
- View grievance history  

### Admin Panel
- Dashboard to view all grievances  
- Update complaint status  
- Respond to submitted issues  
- Manage users and system activity  

### Investigator Panel
- Access assigned grievances  
- Analyze complaints and add investigation notes  
- Update case progress  
- Assist in resolution workflow  

---

## Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: Flask (Python)  
- Database: MySQL  
- Version Control: Git and GitHub
